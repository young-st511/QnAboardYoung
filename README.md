## Description of Project

### Why?
현재 옥소폴리틱스의 QnA 게시판이 Notion으로 만들어져 있습니다. <br/>
저희는 Notion 대신 옥소폴리틱스의 웹 서비스로 구현을 하였습니다.

### What?
<a href="https://faq.oxopolitics.com/">옥소폴리틱스 QnA</a>의 노션으로 디자인 된 Q&A를 웹페이지로 제작하였습니다.
